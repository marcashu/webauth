HTML 4 vs. HTML 5:

One of the differences between HTML 4 and HTML 5 is the syntax, it has been simplified and made easier to use and understand, for example the tags there are many tags which have changed add and even some removed in order to make functions easier to use and create new function too, for example the a tag which previously was used as an anchor as well as for hyperlinking it is now only used for hyperlinking. Another example of the differences between HTML4 and HTML5 is that in HTML4 the syntax was actually less strict as you didn’t necessarily need to close all tags, which is actually a good thing initially but eventually you come to the realisation that you need to close tags due to compatibility issues with other browsers.

Another difference between the two is that it has been made much easier to gain access of the user’s geographical location as in HTML 4 it was very difficult but the new implementation of the Geolocation using JavaScript makes it so much easier in HMTL5.

A big improvement which came with HTML5 is the support of integration of multimedia elements into webpages which allow developers to add videos and audios into their pages whereas on HTML4 there would be a plugin from a third party required which would have to be installed by the user an example would be something such as flash.

Web Standards:

Web standards are the rules and guidelines which a web developer is required to follow when developing a web page, earlier I discussed the differences between some of the standards in HTML4 and HTML5. These web standards are to be followed when creating and designing a website in order for consistency in the design of code in order for there to be a systematic way in which a website is presented to a user.

Some of the advantageous elements of these standards are that the web pages can be accessed from a variety of different browsers and devices which is known as interoperability. Another great advantage that comes with having a fully validated website is that it makes it easier to maintain your website which is due to the standards need for clean code.
